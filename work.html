EcoBigi

Inzamelmachine

EcoBigi is a smart waste-collection machine built using an Arduino microcontroller. Designed and developed as a school project at UNASAT, it detects and sorts recyclable materials, encouraging sustainable habits in everyday environments.

The Challenge

Recycling rates in schools are low because the process feels tedious. We wanted to make sorting fun, automatic, and rewarding.

My Role

Programming the Arduino logic, wiring sensors, and designing the detection algorithm that identifies material types.

  • Arduino C++
  • Servo Motors
  • IR Sensors
  • Ultrasonic Sensor
  • LCD Display
