VIDEO INTEGRATION
=================

To add your video to the About page:

1. Place your video file in THIS folder (assets/video/)
2. Name it:  demo.mp4
3. Reload about.html in Live Server

The player will appear automatically — no code changes needed.

Supported formats: MP4 (recommended), WebM, OGG

If you want to use a different filename, open about.html and find:
    <source src="assets/video/demo.mp4" type="video/mp4" />
and change "demo.mp4" to your filename.
